Help & Risk Disclosures

Real-Money Failure Modes

  • Browser tab suspension (mobile/desktop) stops tick processing — open contracts may remain unmanaged.
  • Network drops can sever the WebSocket mid-trade; reconnection may miss contract expiry.
  • Deriv API server-side disconnects are common; always check open contracts after reconnect.
  • This bot cannot guarantee order execution — proposals expire, prices move, buy requests can be rejected.
  • Past performance of indicator weights does NOT guarantee future results on R_50 or any synthetic.

Before Going Live

  • Run in Shadow Mode for at least 50 simulated trades before enabling real execution.
  • Use the minimum stake your account allows when starting live.
  • Set daily loss limit below 5% of account balance.
  • Keep the browser tab active and screen on — do not rely on background execution.
  • Verify your token has only "Trade" scope, not "Withdraw" scope, as a safety measure.

Early Sell / Contract Limitations

  • Not all Deriv contract types support early sell. Rise/Fall (CALL/PUT) generally do, but sell availability depends on time remaining and market conditions.
  • The sell price shown is indicative — actual execution price may differ.
  • If is_valid_to_sell is false, the sell button is disabled. This is intentional.
  • Early sell locks in a partial profit or loss — weigh this against letting the contract expire.

Browser Reliability Limitations

  • Mobile browsers aggressively suspend background tabs — keep this tab in the foreground.
  • Do not run multiple bot instances with the same token simultaneously.
  • Browser memory limits can cause crashes on very long sessions; reload periodically.
  • No state is persisted across page reloads by design — trade journal is session-only.

Strategy Adaptation Limitations

  • Adaptive weights require a minimum sample size (20 trades per regime) before meaningful adjustment.
  • Weight drift is capped at ±0.15 per adaptation cycle to prevent overfitting.
  • R_50 is a synthetic index — it does not follow real-market fundamentals. Price action patterns may differ from real assets.
  • The regime classifier (ADX + volatility) is a heuristic, not a guarantee.
  • Confidence scores are derived from indicator agreement, NOT from an AI model.
